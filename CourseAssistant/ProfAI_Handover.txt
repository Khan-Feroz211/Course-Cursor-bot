Prof. AI Assistant – User Guide & Handover

---

System Requirements
-------------------
- Windows 10 or 11 (64-bit)
- Python 3.11 (do not use 3.13)
- Ollama (AI model runtime): https://ollama.com/
- Internet connection (for first-time setup)

Installation Steps
------------------
1. Install Python 3.11: https://www.python.org/downloads/release/python-3110/
2. Install Ollama: https://ollama.com/
3. Unzip or copy the `CourseAssistant` folder to your computer.

How to Start the App
--------------------
1. Open the `CourseAssistant` folder.
2. Double-click `Launch_Windows.bat`.
3. Wait for the browser to open at http://localhost:8000 (first time may take a few minutes).
4. Log in with the credentials below.

Troubleshooting Tips
--------------------
- If the browser does not open or you see “localhost refused to connect”:
  - Make sure Python and Ollama are installed.
  - Ensure no other app is using port 8000.
  - Try restarting your computer and running `Launch_Windows.bat` again.
  - If you see errors about missing dependencies, run `venv\Scripts\Activate.ps1` and then `pip install -r requirements.txt` in the `CourseAssistant` folder.
